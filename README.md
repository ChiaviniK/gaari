## 🐶 Sobre o Projeto

Este projeto é uma **Landing Page** responsiva desenvolvida para o **GAARI** (Grupo de Apoio aos Animais da Rua de Iguape), uma organização dedicada a proteger, resgatar e transformar a vida de cães e gatos em situação de rua. 

O foco principal do site é aumentar a visibilidade da ONG e facilitar doações por meio do Pix, ajudando na captação de recursos para cuidados veterinários, ração e abrigo.

## ✨ Funcionalidades

- **Design Responsivo**: Adaptado para uma leitura perfeita em celulares, tablets e computadores.
- **Micro-interações**: Animações suaves de CSS (hover e fade-in) e efeitos modernos como *Glassmorphism*.
- **Cópia de Chave Pix Automática**: Um clique copia a chave CNPJ para a área de transferência do doador.
- **Scroll Suave**: Navegação limpa e fluída através das âncoras da página principal.

## 💻 Tecnologias

A aplicação foi desenhada mantendo a máxima flexibilidade, alta performance e suporte nativo ao **GitHub Pages**, sem necessidades de *builds* complexos ou *frameworks* pesados.

- **HTML5**: Semântica robusta para SEO.
- **CSS3 (Vanilla)**: Design visual usando variáveis CSS e responsividade inteligente.
- **JavaScript**: Lógica de interação na página (Scroll e Clipboard API para a cópia do Pix).

## 🚀 Como Usar

Para visualizar ou modificar o projeto localmente, siga estes passos muito simples:

1. Clone o repositório ou baixe a pasta do projeto.
```bash
git clone https://github.com/seu-usuario/gaari-site.git
```
2. Abra a pasta do projeto `site/`.
3. Dê um clique duplo no arquivo `index.html` e ele abrirá perfeitamente no seu navegador padrão.

*(Opcional: você pode usar a extensão "Live Server" do VSCode para ter auto-reload das páginas durante o desenvolvimento).*
